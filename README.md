## 📝 What is this?

A clean, minimal **To-Do List App** to manage daily tasks — add, complete, filter, and delete tasks with tasks **saved automatically** using `localStorage`, so your list stays even after closing the browser.

---

## ✅ Features

- ➕ **Add Tasks** — quickly type and press enter
- ✔️ **Mark Complete** — strike-through done tasks
- 🗑️ **Delete Tasks** — remove what's no longer needed
- 🔍 **Filter Tasks** — All / Active / Completed views
- 💾 **Persistent Storage** — tasks saved via `localStorage`
- 📱 Fully responsive on mobile & desktop

---

## 🧮 How It Works

```javascript
addTask()      →  Creates new task & saves to localStorage
toggleTask()   →  Marks task complete / incomplete
deleteTask()   →  Removes task from list & storage
filterTasks()  →  Shows All / Active / Completed
```
