📝 To-Do App


📌 Overview


A simple and user-friendly To-Do App that helps users manage their daily tasks efficiently.  
Users can add, complete, and delete tasks to keep track of their work.



✨ Features


- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 📋 Manage daily activities easily
- ⚡ Simple and clean user interface

  

🛠️ Tech Stack


- HTML
- CSS
- JavaScript

  

🎯 Usage


This To-Do App can be used to organize daily tasks, create a task list, and improve productivity.



🔮 Future Improvements


- Add task editing feature
- Add local storage support
- Add dark mode
